# Welcome to your Expo app 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Preparation

1. Firebase setup

    Get google-services.json and GoogleService-Info.plist and put it inside `secrets` folder


## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Build the app (since it uses CNG)

   ```bash
   yarn prebuild
   ```

3. Run the app on iOS Simulator

    ```bash
   npx expo run:ios
   ```
   
4. Run the app on Android Emulator

    ```bash
   npx expo run:android
   ```
